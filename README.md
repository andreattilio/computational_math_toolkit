# Computational Math Toolkit

## Overview
A library of interactive mathematical engines and visualization tools built in the **Wolfram Language**. This project demonstrates the application of **symbolic computation** and **functional programming** to solve geometric and statistical problems.

## Modules

### 1. Geometry & Solver Engines (`/solvers`)
* **Triangle Solver:** A robust logic engine using `Associations` (Key-Value pairs) to handle ambiguous geometric inputs (e.g., Sine/Cosine rules).
* **Polygon Transformation:** Algorithms for geometric enlargement using `ScalingTransform` and vector logic relative to arbitrary center points.

### 2. Interactive Visualizations (`/visualizations`)
* **Calculus Explorer:** Dynamic `Manipulate` interfaces for the Fundamental Theorem of Calculus and Tangent Line gradients.
* **Statistical Dashboard:** A `DynamicModule` interface for frequency distribution analysis and real-time barchart updates.
